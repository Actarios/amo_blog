# Установка
> Необходимо иметь локально `Node 16+`, `npm 8+`
Копируем репозиторий в нужную директорию и переходим в неё.
```sh
$ git clone https://github.com/Actarios/amo_blog.git your_folder && cd your_folder
```
Устанавливаем зависимости.
```sh
$ npm i
```
# Запуск проекта
```sh
$ npm start
```
